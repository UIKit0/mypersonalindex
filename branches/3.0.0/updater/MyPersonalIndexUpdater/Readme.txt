Hi, thanks for using MyPersonalIndex!  To upgrade to the new version 3.0, perform the following steps:

1. Install version 3.0 or above.
2. Open the new version, which should show a welcome message.
3. Now close out.
4. Run MyPersonalIndexUpdater.bat in this folder.
5. It should show "Upgrade Successful".
6. Reopen the new version to view your existing data. Click updates prices to start calculating.
